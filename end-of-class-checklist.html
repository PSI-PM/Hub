Class Management

End-of-Class Checklist

Two things to confirm within a day or two after the last class session. Same process for every course (PMP, CAPM, Practical, CPMAI).

⏱️

Timing: complete both steps within 1–2 days of the last class session. There's no other end-of-class admin beyond what's listed here — no student email, no billing step, no LMS wrap-up.

1
Always

Verify transcripts & certificates generated

Submitting attendance on the last day of class automatically triggers transcript and certificate generation. Confirm it actually went through.

  • Go to Training Records → the class's folder.
  • Confirm the transcript and certificate files are present for the class.
If they're missing: this means the automated pipeline didn't run correctly — flag it as a tooling issue in Step 2 below, since that's exactly the kind of process problem the debrief is for.
2
Only if something happened

Send a debrief to Roy and/or Scott

This step is conditional — skip it entirely if the class ran smoothly with no tooling or process issues. If something did come up, email Roy and/or Scott with the details.

This covers any issue with:

Slides LMS Videos Quizzes Attendance Portal Any other tool or process
Why this matters: this is the feedback loop for everything PSI has built — slide decks, the LMS setup, the Attendance Portal, transcripts. If it broke or caused friction, this is how it gets fixed.
