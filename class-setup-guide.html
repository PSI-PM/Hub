Class Setup Guide

The standard process for getting a class ready to launch — from the roster pull six days out through day-of escalation if a student can't be reached. This process is the same for every course (PMP, CAPM, Practical, CPMAI).

📅

Timing assumption: this guide assumes a Monday class start, which is the standard PSI schedule. "Day 6" through "Day 5" below count backward from that Monday — adjust the calendar dates if a class starts on a different day, but keep the same number of days of lead time.

6 days before class

Pull the roster & kick off prep

  • Go to Training Records in SharePoint → click your class name → click Student Files. This is the simplified view with student names and emails.
  • Email that roster to the LMS Administrator (Amy Hamilton) so she can upload the students into the correct course in the LMS.
  • Verify you have the current version of the slide deck for this course.
  • Create the Zoom meeting using your own Zoom account.
Don't forget: add Roy and Scott as Co-Hosts on the Zoom meeting for every class, so either can step in to teach in an emergency.
5 days before class — deadline

Send the Welcome Email & calendar invite

  • Send the Welcome Email first. Start from the template in the Student Communication folder — personalizing it is fine.
  • Separately, send the Outlook calendar invite for the Zoom session. Don't rely on the email alone for the Zoom link — the calendar invite is what lets you track acceptance.
  • On both the email and the calendar invite: put Roy and Scott in CC, and put all students in BCC.
See the privacy note below — this BCC step is required, not optional.
Days 4–2 before class

Track acceptance & send reminders

  • Each day, check Outlook for who has accepted the calendar invite (a quick visual check — no separate tracking sheet needed).
  • For anyone who hasn't accepted yet, send a short daily reminder — just a nudge to accept, not a full re-send of class details.
Day before class (Sunday)

Escalate non-responders

  • Email Roy and/or Scott the list of students who still haven't accepted or responded.
Day of class (Monday)

Phone follow-up

  • Roy, Scott, or an Admin Manager will personally call the students who haven't responded, to try to make contact before/as class starts.
  • If a student still can't be reached and doesn't show up, the instructor documents that outreach was attempted when informing the customer point of contact — note that PSI tried to reach the student and they didn't show.

🔒 Privacy requirement — every time

Students must never be able to see another student's name or email address. This applies to the Welcome Email and the Outlook calendar invite, with no exceptions.

  • Students go in BCC — never To, never CC.
  • Roy and Scott go in CC — they're staff, so visibility between them and the instructor is fine.
